# Zara AI Companion

Zara is a memory-aware conversational guide inspired by warmth, reflection, and grounded wisdom.

This service provides a public text-to-text chat experience with:
- Persistent per-user memory
- Emotional awareness
- Long-term personalization
- Safe public deployment

Built with Node.js and OpenAI, designed for scalable hosting.

This repository contains the server-side application only.
